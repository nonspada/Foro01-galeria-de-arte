Sobre la página

Está página web fue hecha en HTML, CSS y un poco de Javascript. Es un único archivo alojado en un repositorio de Github y desplegado en Github Pages.

Su finalidad es ayudar comprender de forma práctica cómo los elementos HTML estructuran una página, cómo CSS aplica estilos a ellos, y cómo se despliega el resultado de ésta para que sea disponible desde una URL en la web.

Para su fabricación se usaron herramientas populares para el desarrollo web, tales como IDE, sistemas de control de versiones (Git), imágenes de Wikipedia e IA generativas.
